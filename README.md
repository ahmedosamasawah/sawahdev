# Portfolio

Personal portfolio and blog built with SvelteKit.

## Stack

- SvelteKit
- Svelte 5
- TypeScript
- Tailwind CSS
- Static adapter for GitHub Pages

## Development

```sh
pnpm install
pnpm dev
```

## Checks

```sh
pnpm lint
pnpm check
pnpm build
```

## Deploy

GitHub Actions builds the static site and deploys `build/` to GitHub Pages.
