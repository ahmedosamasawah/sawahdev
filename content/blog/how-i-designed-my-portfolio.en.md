# How I Designed This Portfolio

Design is storytelling.

## Typography
I started with Inter — readable, elegant, and neutral.  
Then added a subtle modular scale for hierarchy.

## Color palette
Dark mode first, because let’s be honest: developers prefer it.

Colors:

- accent: a soft purple  
- foreground: high-contrast white  
- card: layered neutrals  

## Motion
I used micro-animations:

- hover effects  
- subtle transitions  
- parallax-like star decorations  

## Layout decisions
The layout aims to balance personality and clarity:

- strong hero sections  
- centered content  
- restrained decoration  

## Closing thoughts
Design is never finished, but this portfolio finally feels like “me”.
