# Hello World

Welcome to my new portfolio and blog.

This is the first post published using my custom **Svelte 5 static site generator** — built from scratch, without SvelteKit, just because I love learning by doing.

## Why I built it

I wanted something that gave me:

- full control
- full transparency
- zero magic
- and a deeper understanding of how SSG pipelines work

I implemented:

- a manifest-driven routing system
- fully static rendering
- automatic OG image generation
- a custom Markdown → HTML compiler
- a multi-locale (EN/AR) blog pipeline

### What’s next?

Expect posts about:

- performance tuning
- TypeScript patterns
- UI polish
- and some more advanced Svelte runes techniques

Thanks for reading — and stay tuned!
